I'm a computer science student at the University of Rochester. I work on backend systems in Python and Java. Previously, I worked as a software engineering intern at AnytimeAI and as a research assistant at the Jiang Lab.

I wrote Mayulime, the fastest (at time of writing) multi-pair shortest path vertex interdiction solver. Currently, I'm working on Serta, a 2D platformer.

I believe the software we craft should be as performant as possible: if you notice it running, it's too slow. Code should be straightforward and boring. Software is naturally complex, but it can be made comprehensible to both humans and coding agents.

I'm particularly interested in expressing domain models with manifest typing and scaling concurrent programs on distributed systems, but I'm happy to work on anything.
